Projeto de simulação de monitoramento de indicadores industriais utilizando PostgreSQL, Python, Excel e Power BI.

Este repositório contém o código-fonte e a documentação do projeto de sistema de monitoramento industrial. O sistema é projetado para coletar, analisar e visualizar dados em tempo real de equipamentos industriais, permitindo a detecção precoce de falhas e a otimização da manutenção.

## Objetivo

Simular coleta, validação e visualização de dados de produção para cálculo de OEE.

## Status

Em desenvolvimento.

## Funcionalidades

- Coleta de dados de sensores industriais.
- Análise de dados para detecção de anomalias.
- Visualização de dados através de dashboards interativos.

## Tecnologias Utilizadas

- Python para coleta e análise de dados.
- PostgreSQL para armazenamento de dados.
- Excel para análise adicional e geração de relatórios.
- Power BI para visualização de dados e criação de dashboards.
